# Arquivos para teste manual de importacao

Use estes CSVs na tela `Importacoes > Files`.

Login local quando a API estiver fora:

```txt
usuario: admin
senha: admin
```

Sugestao de uso:
- `vendas-notas-fiscais-demo.csv`: testa destino automatico `Vendas (Notas Fiscais)`.
- `clientes-cadastro-demo.csv`: testa destino automatico `Clientes`.
- `produtos-catalogo-demo.csv`: testa destino automatico `Produtos`.
- `pedidos-demo.csv`: testa destino automatico `Pedidos`.
- `vendas-notas-fiscais-com-erros-demo.csv`: testa campos faltando, valores negativos e datas invalidas.

Os arquivos sao pequenos de proposito para voce conseguir alterar valores, remover colunas e simular erros rapidamente.
